# d3-geo-example
An example project for d3 geo both v3 and v4

### RUN

```
mvn spring-boot:run

```

### V3 version url

```
http://localhost:8080/d3-geo.v3.html

```

### V4 version url

```
http://localhost:8080/d3-geo.v4.html

```
